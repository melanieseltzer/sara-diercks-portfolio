# Sara Diercks

> The GIS portfolio of Sara Diercks 🌎

https://sara-diercks.netlify.com/

Showcasing projects from UCLA Extension's GIS and Geospatial Technology certificate program.

## Development 🚀

Start the dev server:

`yarn dev`

Build for production:

`yarn build`

Start the app:

`yarn start`
